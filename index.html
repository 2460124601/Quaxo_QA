Quaxo Diffusion常見問題

申請資格

申請流程如下：

  1. 申請Quaxo ID
  2. 透過驗證電子郵件啟用帳號
  3. 登入您的帳號
  4. 您可透過管理面板中，右側部分送出申請表單
  5. 靜待申請結果，審核約1~12小時

下載安裝包

當您取得測試資格後，您可透過管理面板下載

  1. 登入管理面板
  2. 於右側同意《使用條款》並下載

安裝插件


  1. 延伸應用程式>擴展程式管理器
    2. 點選圖片可以進入全螢幕模式
  2. 點選「安裝擴充程式」
    3. 點選圖片可以進入全螢幕模式
  3. 開啟檔案管理員後，即可點選Quaxo插件安裝包（.rbz格式）即完成安裝

使用教學

  1. 安裝完成後，您可以看見蝴蝶結圖示，點選後即可開啟控制面板
    2. 點選圖片可以進入全螢幕模式
  2. 開啟後，請登入Quaxo ID帳號
  3. 進入生成面板，需要先調好生成模型之角度，接下來輸入提示詞，或是選擇預設集(兩者不衝突)
  4. 送出後，請靜待15~30秒
  5. 生成完成，您即可下載，或是選擇重新生成
  6. 生成效果對比

    A：因為部分版本無法支援較新的介面程式語法，導致，但這不會影響功能使用。
