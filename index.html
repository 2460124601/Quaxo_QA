什麼是「網頁前端Front-End）」？

網頁前端泛指用户在瀏覽器中看到並與之交互的部分，可以由文字、圖片、多媒體構成。主要目的是顯示資訊、提供後端程式交互的平台。


如何構成？

在瀏覽器前端的世界中，主要是由：HTML、CSS、JavaScript組成。

  • HTML：超文本標記語言。主要作用於定位元素，例如：標題、圖片、文本。是一個網頁最基本且必須的骨架。
  • CSS：用於設計和控制網頁的外觀和佈局。CSS可以設定文字的字體、顏色、大小，背景的顏色和圖像，元素的位置等。簡單來說：就是幫網頁「上妝」。
  • JavaScript：用於為網頁添加動態行為和互動功能。隨著技術日新月異，瀏覽器的API逐漸增加以及調用的權限越來越豐富，JavaScript能做到的效果、功能不亞於桌面應用。

HTML規範

W3C是HTML規範的主要維護組織。由Tim Berners-Lee於1994年創立，旨在制定Web技術的標準，確保Web技術的長期發展和互操作性。


基本語法

在HTML中基本語法是使用<XXX>包裹標籤名，結尾則增加斜線在標籤前，如：<XXX>內容</XXX>

階層樣式表

階層樣式表（CSS，Cascading Style Sheets）是一種用於描述HTML或XML文檔的呈現方式的語言。它允許你以分層方式控制網頁的樣式和佈局，包括顏色、字體、間距、佈局等。CSS 是網頁設計的核心技術之一，與 HTML 一起使用時，可以將內容和樣式分離，從而提高網頁的可維護性和一致性。

元素選擇器（Type Selector）

/* 用於選擇HTML內建的標籤 */
h1{
    color:red;
}

類選擇器（Class Selector）

/* 用於選擇class */
.productBox{
    border-radius:10px;
    box-shadow: rgba(0, 0, 0, 0.35) 0px 5px 15px;
}

ID選擇器（ID Selector）

/* 用於選擇ID */
#UserInfo{
    font-size:20px;
}
